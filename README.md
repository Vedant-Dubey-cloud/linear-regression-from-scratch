# Linear Regression From Scratch

This project implements Ordinary Least Squares (OLS) Linear Regression from scratch using NumPy without relying on machine learning libraries for training. The model is trained using the Normal Equation and provides functionality similar to scikit-learn's `LinearRegression`, including coefficient estimation, intercept calculation, prediction, and performance evaluation using the R² metric.

## Features
- Linear Regression implemented from scratch
- Normal Equation based training
- Custom `fit()` and `predict()` methods
- Automatic intercept handling
- R² score evaluation
- Validation against scikit-learn
